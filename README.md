
# DevFrame

A creative personal portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, and no heavy dependencies — just raw frontend code focused on design, interaction, and personality. [cite:25]

This is a personal portfolio project created to showcase my skills, projects, and learning journey as a BCA student and aspiring full-stack developer. It is built as a fun and experimental space where I can combine web development, interactivity, and creativity in one place. [cite:25][cite:1]

## About This Project

This portfolio reflects my interest in modern frontend design, interactive experiences, and developer-focused storytelling. Instead of using large frameworks, I wanted to create something lightweight and personal using core web technologies. [cite:25]

The project highlights my work, skills, journey, and contact details through a visually bold design. It also includes playful interactions and portfolio sections that make the site feel more engaging than a basic resume page. [cite:25]

## Design Style

The website uses a bold and playful visual style inspired by neo-brutalism and hand-crafted UI ideas. It combines strong borders, flat shadows, high-contrast accents, and creative section layouts to make the portfolio stand out. [cite:25]

Key visual ideas include: [cite:25]
- **Thick borders** for a strong and structured look. [cite:25]
- **Flat offset shadows** to create a comic-card effect. [cite:25]
- **Bright highlight colors** like yellow, cyan, pink, and green for emphasis. [cite:25]
- **Hand-crafted visual touches** that make the site feel more personal and creative. [cite:25]
- **Interactive sections and animations** that improve the browsing experience. [cite:25]

## Design Decisions

### Color System

The site uses CSS custom properties for theming and reusable design tokens. The palette is based on bold accent colors with black borders and clean surfaces. [cite:25]

| Color | Variable | Usage |
|---|---|---|
| Yellow [cite:25] | `--yellow` [cite:25] | Main highlight, CTA areas |
| Cyan [cite:25] | `--cyan` [cite:25] | Code-style accents, highlights |
| Pink [cite:25] | `--pink` [cite:25] | Secondary highlight accents |
| Green [cite:25] | `--green` [cite:25] | Positive accents, emphasis |
| Black [cite:25] | `--border` [cite:25] | Borders, shadows, text |

### Typography

The portfolio uses modern fonts that support both clean UI and developer aesthetics. Monospace fonts are used for code-inspired or terminal-inspired parts, while the main font keeps the site readable and bold. [cite:25]

### Key Design Elements

**Hero Section** — A visually strong landing section that introduces me, my role, and the technologies I work with. It is designed to feel friendly, creative, and developer-focused. [cite:25][cite:1]

**Journey Section** — A section that presents my academic and development journey, including my growth as a BCA student and builder of real-world projects. [cite:25]

**Skills Section** — A clean overview of the technologies I use, such as JavaScript, React, Next.js, Tailwind CSS, Node.js, MongoDB, and related tools. [cite:1]

**Projects Section** — A showcase of the work I have built, including portfolio websites, chat apps, quiz platforms, games, galleries, and other experimental web projects. [cite:1]

**Interactive Elements** — Scroll effects, hover states, highlights, and animations help make the site more dynamic and memorable. [cite:25]

## Terminal Mode

The portfolio can also include a terminal-style resume view for a more unique developer experience. This mode presents information in an interactive terminal interface instead of a regular portfolio layout. [cite:25]

Features may include: [cite:25]
- Type `help` to see available commands. [cite:25]
- View sections like about, skills, projects, education, and contact through commands. [cite:25]
- Multiple terminal themes. [cite:25]
- Command history support. [cite:25]
- Resume PDF download support. [cite:25]
